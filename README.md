# PianoPro
PianoPro is a professional piano learning and MIDI visualization tool, inspired by Guitar Pro but designed specifically for piano.

It allows you to load MIDI files and interact with them through a clear, structured, and practice-oriented interface.  
The goal is to provide a more focused and less flashy experience, better suited for serious learning and analysis.

## Features

- MIDI file playback with synchronized piano visualization  
- Clear note representation for easier reading and practice  
- Practice-oriented interface (tempo control, track selection, etc.)  
- Designed for learning without requiring deep music theory knowledge  

## Philosophy

This project aims to bring a more structured and professional approach to piano learning using MIDI, similar to what Guitar Pro provides for guitar players.
