# Web App

Use PianoPro directly in the browser.

[Open PianoPro Web](/app/){target="_self"}

## Included

- MIDI import and playback
- Browser-persisted MIDI library
- Continue flow after refresh
- Built-in Web Audio playback
- Main menu, settings, free play, and playback scenes shared with the desktop app

## Current limitations

- Audio import remains desktop-only.
- Custom SoundFont selection remains desktop-only.
- Best results are on a recent desktop browser.
- Large MIDI libraries may take longer to restore after a refresh.
